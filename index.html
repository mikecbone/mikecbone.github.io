Unity 2D

In my own time I learnt Unity and C# and with this I created a simple 2D platformer.

The project was source controlled using GitKraken Pro and you can view the finished project on GitHub here.

Finally, you can play the game here and attempt to complete all 4 levels and collect all 50 coins. Alternativly hit backspace to jump to the finish screen and pretend you won!

ChessoR

A Chess iPhone application created in Swift that involved the use of Artificial Intelligence and search algorithms to create an AI opponent of varying difficulty, along with loading and saving games, support for special moves such as pawn promotion, hot swapping games and saved user statistics.
